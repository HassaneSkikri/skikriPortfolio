

# Portfolio Project

## Installation Guide

### install the React app

```npx create-react-app my-app```	

### Core Functionality
1. **Bootstrap**
   - **Command**: `npm install bootstrap`
   - **Utility**: Installs Bootstrap for responsive and mobile-first site development.
   
2. **React-Bootstrap**
   - **Command**: `npm install react-bootstrap`
   - **Utility**: Provides Bootstrap components as React components.

3. **React Router Hash Link**
   - **Command**: `npm install react-router-hash-link`
   - **Utility**: Adds smooth scrolling for anchor links in React Router.

4. **React Router DOM**
   - **Command**: `npm install react-router-dom`
   - **Utility**: Enables dynamic routing in a web application.

### Banner Section
5. **React Bootstrap Icons**
   - **Command**: `npm install react-bootstrap-icons`
   - **Utility**: Adds Bootstrap icons as React components.

6. **Animate.css**
   - **Command**: `npm install animate.css`
   - **Utility**: Incorporates CSS animations easily into HTML elements.

7. **React On Screen**
   - **Command**: `npm install react-on-screen`
   - **Utility**: Detects if a component is visible in the viewport for triggering animations or actions.

### Skills Section
8. **React Multi Carousel**
   - **Command**: `npm install react-multi-carousel`
   - **Utility**: Provides a responsive carousel for displaying multiple items, like images or content sliders.
